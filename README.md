# Conselho
## Projeto em Kotlin que gera um conselho aleatorio após clicar no botão
### Tecnologias utilizadas:
- Retrofit
- Coroutines 
- Gson
- LifeCycle
- LiveData
- ViewModel
- Koin
- Arquitetura MVVM
